# Banking operations

## Описание проекта:
```
Это виджет, который показывает несколько последних успешных 
банковских операций клиента 
(фильтрация и сортировка банковских счетов по дате и оплате).
```

```
- Функции маскировки номера банковской карты и счета

```
